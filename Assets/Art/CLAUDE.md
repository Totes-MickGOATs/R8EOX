# Art

## Purpose
Visual assets: textures, 3D models, sprites, and animations.

## Conventions
- PascalCase with type suffix for textures: `PlayerSkin_Albedo.png`, `Ground_Normal.png`
- Organize in subfolders: `Textures/`, `Models/`, `Sprites/`
- Import settings should be configured per asset type (compression, filtering, etc.)
- Keep source files (PSD, FBX) alongside their exported versions
