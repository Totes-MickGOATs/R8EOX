## Purpose
Regression tests auto-generated by the E2E quality agent. Each test guards against a specific bug that was found and fixed.

## Conventions
- Namespace: `R8EOX.Tests.PlayMode`
- Category: `[Category("regression")]`
- One test class per bug, named after the issue (e.g., `VehicleSpawnHeightRegressionTests.cs`)
- Each file should document the original bug in a comment at the top
- Tests should be minimal — verify the specific fix, not the entire flow
